# iddiligence-demo

Demo Application for testing IDDILIGENCE, our solution for validation of data, documents and identity for digital customers.

## Dependencies
- Angular.js
- Angular-Route
- Angular-Resource
- Bootstrap
- Fontawesome

## Usage
### Install global
```
$ npm install -g stylus
$ npm install -g gulp
$ npm install -g bower
$ npm install â€“g karma
$ npm install -g protractor
$ npm install -g karma-coverage
$ npm install -g karma-junit-reporter

```

### Install local
```
$ npm install
$ bower install
```
### Development mode
```
$ gulp
$ gulp build-debug
```
### Go-to-production
```
$ gulp build
$ gulp server-dist
```

### Generate war distribution
```
$ gulp target
```
